Chapter 10 Default settings

Some of the default settings of scan can be changed with the options() argument. Table 10.1 shows a complete list of options and their default values.

# get the current value of an option
getOption("scan.print.rows")
[1] 15
# set option to a different value
options(scan.print.rows = 5, scan.print.scdf.name = FALSE)
print(exampleAB)

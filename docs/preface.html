Software reference

This book has been created using the Rmarkdown (Allaire et al., 2021) and bookdown (Xie, 2020) packages within the RStudio (RStudio Team, 2018) environment. The analyses have been conducted with the R package scan at version 0.53 (Wilbert & Lueke, 2021). R version 4.0.4 (2021-02-15) was used (R Core Team, 2021).
